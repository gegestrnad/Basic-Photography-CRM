Data Backup & Restore
Local File Backup
Traditional download/upload — works offline
Cloud Backup (Share to Drive, OneDrive, Dropbox…)
Saves to Downloads, then opens Android share sheet — pick Google Drive, OneDrive, Dropbox, etc.

Share uses Android's native share sheet. Load picks from Downloads/Drive/Files app.

Export to Excel / CSV (for Google Sheets)
Creates a multi-sheet Excel file — open in Excel, Google Sheets, LibreOffice

For Google Sheets: export Excel → open Google Sheets → File → Import → Upload the .xlsx

Google Sheets Sync
Sync with Google Sheets
Push data to a live Google Sheet — readable by anyone you share the sheet with.

Paste the URL from: Extensions › Apps Script › Deploy › Web app

When enabled, every save silently pushes to Sheets in the background.

Setup Guide
  1. Open (or create) a Google Sheet
  2. Click Extensions › Apps Script
  3. Paste the contents of google-apps-script.js into Code.gs — replace everything
  4. Click Deploy › New deployment › Web app
  5. Set Execute as: Me and Who has access: Anyone
  6. Copy the Web App URL and paste it above
  7. Click Save Config, then Push Now to test
